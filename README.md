## ⚙️ INSTALLATION

### 1. Install Dependencies
```bash
cd discord-phone-tracker-bot
npm install
```

### 2. Configure Bot
Edit `.env` file:
```env
BOT_TOKEN=YOUR_DISCORD_BOT_TOKEN
PREFIX=!
```

### 3. Start Bot
```bash
npm start
```

---

## 🔧 DISCORD BOT SETUP

1. Go to https://discord.com/developers/applications
2. Create New Application
3. Go to Bot Section -> Create Bot
4. Copy Bot Token
5. Enable **Message Content Intent** & **Server Members Intent**
6. Invite Bot to your server with Administrator Permissions
